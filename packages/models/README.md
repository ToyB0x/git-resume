## 概要

このパッケージは、git-resumeプロジェクトで使用される共通データモデルを提供します。各アプリケーション（API、CLI、Web）で一貫したデータ構造を維持するために使用されます。

## 目的

アプリケーション間でのデータ構造の一貫性を確保し、型安全な開発環境を提供することで、開発効率と品質を向上させることを目的としています。

## 主な機能

- GitHubユーザー情報のモデル定義
- GitHubリポジトリ情報のモデル定義
- レジュメデータのモデル定義
- リポジトリパッケージング情報のモデル定義
- サマリー情報のモデル定義
